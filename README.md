# BND Labs Portfolio

A high-performance, responsive, 3D interactive portfolio for BND Labs built with Vite, React, React Three Fiber, GSAP, and Tailwind CSS. The portfolio acts as a digital proof-of-work asset for a digital agency founder based in Zambia.

## Features
- Fully interactive 3D elements powered by React Three Fiber and GSAP animations.
- Scroll-based section navigation.
- Extensively optimized for performance (lazy loading, code splitting, WebP image formats, and font preloading).
- Excellent technical SEO (structured data, Open Graph configurations, sitemap).
- High-converting CTA forms and contact sections.

## Technologies Used
- React 18
- Vite
- Tailwind CSS
- Three.js / React Three Fiber / React Three Drei
- GSAP
- EmailJS

## Local Setup
1. Clone the repository.
2. Run `npm install` to install all dependencies.
3. Rename `.env.example` to `.env.local` and substitute the EmailJS keys with your own.
4. Run `npm run dev` to start the local development server on `localhost:5173`.

## Build & Deploy
1. Run `npm run build` to generate the highly optimized production bundle.
2. You can preview the built site using `npm run preview`.
3. The application is configured to deploy directly to Vercel via the `vercel.json` file.
